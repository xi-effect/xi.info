# Что прикрепить к уроку на доске: PDF, картинка, звук

Помимо рисования, на доску можно добавлять файлы: изображения, PDF-документы и аудио. Каждый добавленный файл становится объектом на поле доски — его можно перемещать, масштабировать и размещать рядом с рисунками.

## Как добавить изображение

1. Откройте доску.
2. В нижней панели инструментов нажмите кнопку **Изображение** (иконка с картинкой).
3. Откроется диалог выбора файла. Выберите файл в форматах JPEG, PNG, GIF, WebP или другом распространённом графическом формате.
4. Изображение появится на поле доски. Потяните за углы, чтобы изменить размер, или перетащите, чтобы переместить.

## Как добавить PDF

1. В нижней панели инструментов нажмите кнопку «**⋮**» (три точки) в правом крайнем положении.
2. Выберите **Загрузить PDF**.
3. Откроется диалог выбора файла. Выберите PDF-документ.
4. PDF появится на доске как многостраничный объект с возможностью листать страницы.

## Как добавить аудио

1. В нижней панели инструментов нажмите кнопку «**⋮**» (три точки).
2. Выберите **Загрузить аудио**.
3. Откроется диалог выбора файла. Выберите аудиофайл.
4. Аудиостикер появится на доске. Нажмите на него, чтобы воспроизвести.

## Особенности

- Все добавленные объекты (изображения, PDF, аудио) — отдельные элементы на поле доски. Их можно перемещать и при необходимости удалять.
- Если доступ к материалу у ученика — «Совместная работа», ученик тоже видит все добавленные файлы.
- Файл загружается с вашего устройства. Очень большие файлы могут загружаться дольше или вызывать ошибку.
