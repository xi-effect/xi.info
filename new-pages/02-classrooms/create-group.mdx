# Как создать группу

Групповой кабинет подходит для занятий сразу с несколькими учениками: он вмещает до 15 участников. Материалы, расписание и оплаты в нём общие для всей группы.

Создать группу может только репетитор.

## Как создать групповой кабинет

1. Откройте раздел **Кабинеты** в боковом меню.
2. На компьютере нажмите кнопку **Создать группу** в правом верхнем углу. На смартфоне — нажмите плавающую кнопку «+» в правом нижнем углу, затем выберите **Создать группу**.
3. Откроется окно **Создание группы**.

4. Введите **Название** группы — это имя будет отображаться в списке кабинетов.
5. Выберите **Предмет** из выпадающего списка.
6. Нажмите **Создать**.

Группа появится в списке кабинетов. После этого вы можете добавить учеников или пригласить их по ссылке.

## Особенности и ограничения

- В одном групповом кабинете может быть не более **15 учеников**.
- Название и предмет задаются при создании.
- После создания группы пригласите учеников через раздел [Групповой кабинет: добавить ученика, пригласить, удалить кабинет](group-classroom-manage.mdx).

## Если что-то не получается

**Кнопка «Создать группу» не видна:**

- Убедитесь, что вы работаете в роли репетитора. Ученики не могут создавать кабинеты.

**Форма не отправляется:**

- Оба поля обязательны. Проверьте, что название введено и предмет выбран.
