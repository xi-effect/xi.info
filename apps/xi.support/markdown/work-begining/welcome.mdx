<AnchorHeading type="h2" id="final">
  Окончание настройки профиля
</AnchorHeading>

В зависимости от того, насколько вы знакомы с инструментами xi.effect, выберите подходящую опцию для ознакомления с платформой.

Вариант “Посмотреть тур по продукту” ознакомит вас с основными элементами интерфейса.

Вариант “Почитать справку” перенаправит на портал поддержки, где можно найти ответы на возникшие вопросы.
