Раздел **Контроль оплат** в боковом меню — главный экран для работы с финансами. Здесь хранится история всех счетов, а репетитор может создавать новые.

## Как открыть

Нажмите **Оплаты** в боковом меню. Откроется страница **Контроль оплат**.

## Вкладки

### Журнал оплат

Здесь отображается список всех счетов — как выставленных, так и уже оплаченных. Для каждого счёта видны: имя ученика (или репетитора), дата, сумма и статус.

Вкладка **Журнал оплат** доступна и репетитору, и ученику.

### Типы оплат

Здесь репетитор может настраивать шаблоны для быстрого создания счетов. Вкладка доступна **только репетитору**.

## Кнопка «Создать счёт на оплату»

Репетитор видит кнопку **Создать счёт на оплату** в правом верхнем углу страницы (на компьютере). Нажав её, можно быстро создать новый счёт.

Эта же кнопка доступна в кабинете на вкладке **Оплаты**.

## Особенности

- Ученик видит только вкладку **Журнал оплат** — вкладка «Типы оплат» для него скрыта.
- Список прокручивается по мере загрузки — новые записи подгружаются автоматически.
- Открыть конкретный счёт можно по ссылке из письма или прямо из списка.

## Если что-то не получается

**Список пустой:**

- Репетитору: создайте первый счёт кнопкой **Создать счёт на оплату**.
- Ученику: счета появятся, когда репетитор их выставит.
