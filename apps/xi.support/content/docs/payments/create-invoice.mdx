Счёт на оплату создаёт репетитор. Он указывает занятия, стоимость и количество, после чего ученик получает уведомление.

Деньги переводятся напрямую, вне сервиса. Сервис фиксирует факт создания и подтверждения оплаты, не проводя транзакции.

## Что понадобится

- Аккаунт репетитора.
- Хотя бы один активный кабинет с учеником или группой.

## Как создать счёт

Открыть форму создания счёта можно двумя способами:

- На странице **Контроль оплат**: нажмите кнопку **Создать счёт на оплату** в правом верхнем углу.
- В кабинете на вкладке **Оплаты**: нажмите **Создать счёт на оплату** в правом верхнем углу вкладки.

Откроется окно **Создание счета на оплату**.

## Заполнение формы

1. **Кабинет** — выберите кабинет (ученика или группу), за которого выставляется счёт.
2. Нажмите **Добавить занятие**, чтобы добавить строку с занятием. На мобильном кнопка называется **Добавить строку**.
3. Для каждой строки заполните:
   - **Занятия** — название (например: «Математика»).
   - **Стоимость** — цена за одно занятие.
   - **Количество** — сколько занятий.
   - **Сумма** — рассчитывается автоматически.
4. При необходимости используйте **Выбрать шаблон** для быстрого заполнения из сохранённых типов оплат.
5. В поле **Комментарий** можно добавить пояснение к счёту.
6. Проверьте итоговую сумму в строке **Итого**.
7. Нажмите **Создать**.

После создания счёт появится в журнале оплат. Ученик получит уведомление.

## Особенности

- Можно добавить несколько строк с разными занятиями в один счёт.
- Если у вас нет кабинетов, кнопка **Создать** будет недоступна — сначала создайте кабинет или дождитесь, пока ученик примет приглашение.
- Шаблоны («Типы оплат») позволяют быстро заполнять повторяющиеся счета.

## Если что-то не получается

**Кнопка «Создать» недоступна:**

- Убедитесь, что у вас есть хотя бы один кабинет. Если список кабинетов пуст, пригласите ученика сначала.

**Ученик не видит счёт:**

- Счёт появляется в журнале оплат ученика автоматически после создания.
