# Условия использования

**Дата последнего обновления:** 1 января 2024 года

## 1. Общие положения

Настоящие Условия использования (далее — «Условия») регулируют отношения между пользователями и платформой Sovlium (далее — «Платформа», «мы», «нас», «наш»).

Используя Платформу, вы соглашаетесь с данными Условиями. Если вы не согласны с какими-либо положениями, пожалуйста, не используйте Платформу.

## 2. Описание сервиса

Sovlium — это платформа для проведения онлайн-уроков, которая предоставляет:

- Видеозвонки для проведения занятий
- Интерактивные онлайн-доски
- Систему расписания уроков
- Контроль оплат и финансовую отчетность
- Другие образовательные инструменты

## 3. Регистрация и учетные записи

### 3.1 Требования к регистрации

Для использования Платформы необходимо:

- Предоставить достоверную информацию при регистрации
- Подтвердить адрес электронной почты
- Создать надежный пароль
- Быть не младше 18 лет или иметь согласие родителей/опекунов

### 3.2 Ответственность за учетную запись

Вы несете ответственность за:

- Безопасность своей учетной записи
- Все действия, совершенные под вашей учетной записью
- Своевременное уведомление о несанкционированном доступе

## 4. Правила использования

### 4.1 Разрешенное использование

Вы можете использовать Платформу для:

- Проведения образовательных занятий
- Обучения и получения знаний
- Профессиональной деятельности в сфере образования

### 4.2 Запрещенное использование

Запрещается:

- Нарушать права интеллектуальной собственности
- Передавать вредоносный контент
- Нарушать права других пользователей
- Использовать Платформу для незаконной деятельности
- Создавать фальшивые учетные записи
