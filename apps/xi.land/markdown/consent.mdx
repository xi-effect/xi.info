# Согласие на обработку персональных данных

**Дата последнего обновления:** 1 января 2024 года

## 1. Общие положения

Настоящее Согласие на обработку персональных данных (далее — «Согласие») предоставляется в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

**Оператор персональных данных:** ООО «Совлиум» (далее — «Оператор»)

## 2. Категории персональных данных

Я даю согласие на обработку следующих персональных данных:

### 2.1 Основные данные

- Фамилия, имя, отчество
- Дата рождения
- Пол
- Гражданство

### 2.2 Контактные данные

- Адрес электронной почты
- Номер телефона
- Адрес места жительства

### 2.3 Образовательные данные

- Уровень образования
- Специализация
- Опыт работы
- Квалификационные документы

### 2.4 Технические данные

- IP-адрес
- Данные о браузере и устройстве
- Информация о посещениях сайта
- Данные о взаимодействии с сервисом

## 3. Цели обработки персональных данных

Персональные данные обрабатываются в следующих целях:

### 3.1 Основные цели

- Предоставление образовательных услуг
- Обработка заявок и регистраций
- Ведение учета пользователей
- Обеспечение безопасности сервиса

### 3.2 Дополнительные цели

- Маркетинговые коммуникации
- Анализ использования сервиса
- Улучшение качества услуг
- Соблюдение правовых требований

## 4. Способы обработки персональных данных

Обработка персональных данных осуществляется:

- С использованием автоматизированных средств
- Без использования автоматизированных средств
- Смешанным способом

**Конкретные операции:**

- Сбор, запись, систематизация
- Накопление, хранение
- Уточнение, обновление, изменение
- Извлечение, использование, передача
- Обезличивание, блокирование, удаление

## 5. Передача персональных данных

### 5.1 Внутренняя передача

Персональные данные могут передаваться между подразделениями Оператора для достижения целей обработки.

### 5.2 Внешняя передача

Персональные данные могут передаваться:

- **Поставщикам услуг:** для технической поддержки и обработки
- **Партнерам:** для предоставления совместных услуг
- **Государственным органам:** при соблюдении правовых требований
- **Третьим лицам:** с вашего дополнительного согласия

### 5.3 Международные передачи

Данные могут передаваться в страны, обеспечивающие адекватную защиту персональных данных, или при наличии соответствующих гарантий.

## 6. Сроки обработки персональных данных

### 6.1 Сроки хранения

- **Данные аккаунта:** до удаления аккаунта пользователем
- **Данные транзакций:** 5 лет с момента последней операции
- **Данные маркетинга:** до отзыва согласия
- **Данные аналитики:** 2 года с момента сбора

### 6.2 Критерии определения сроков

Сроки определяются:

- Целями обработки данных
- Правовыми требованиями
- Техническими возможностями
- Бизнес-потребностями

## 7. Права субъекта персональных данных

### 7.1 Основные права

Вы имеете право:

- **Получать информацию** о обработке ваших персональных данных
- **Требовать уточнения** персональных данных
- **Требовать удаления** персональных данных
- **Ограничивать обработку** персональных данных
- **Возражать против обработки** персональных данных
- **Требовать прекращения обработки** персональных данных

### 7.2 Порядок реализации прав

Для реализации прав:

1. Обратитесь к нам по email: privacy@sovlium.ru
2. Укажите конкретное право, которое хотите реализовать
3. Предоставьте документы, подтверждающие вашу личность
4. Дождитесь ответа в течение 30 дней

## 8. Отзыв согласия

### 8.1 Право на отзыв

Вы имеете право отозвать настоящее Согласие в любое время.

### 8.2 Последствия отзыва

При отзыве Согласия:

- Мы прекратим обработку ваших персональных данных
- Ваш аккаунт может быть заблокирован
- Некоторые сервисы станут недоступны
- Данные будут удалены в соответствии с политикой хранения

### 8.3 Порядок отзыва

Для отзыва Согласия:

1. Отправьте заявление на email: privacy@sovlium.ru
2. Или воспользуйтесь формой в настройках аккаунта
3. Дождитесь подтверждения отзыва

## 9. Защита персональных данных

### 9.1 Технические меры

- Шифрование данных при передаче и хранении
- Использование защищенных каналов связи
- Регулярное обновление систем безопасности
- Мониторинг и логирование доступа

### 9.2 Организационные меры

- Ограничение доступа к персональным данным
- Обучение сотрудников правилам обработки
- Назначение ответственного за обработку
- Регулярные аудиты безопасности

## 10. Ответственность

### 10.1 Ответственность Оператора

Оператор несет ответственность за:

- Соблюдение требований законодательства
- Обеспечение безопасности персональных данных
- Своевременное уведомление о нарушениях
- Возмещение ущерба при нарушении прав

### 10.2 Ответственность субъекта

При предоставлении недостоверных данных субъект несет ответственность в соответствии с законодательством.
